# CAB202
CAB202 Assessment for Sem2 2015

NOTE: Both assignments require the use of a custom external library that is not provided in this repository. This library was used for graphics and timer functions. Futhermore, Assignment 2 contains a usb_serial library provided by the CAB202 teaching team.


## Assignment 1: Megamaniac 

The task was to make a Megamaniac clone with a given specification. Contains 5 levels of various difficulties 

## Assignment 2: Falling Faces

The task was to write a "Falling Faces" game to be run on a teensy board. This game was also made to a given specification
